.. _Concepts::

Concepts
========

Learn about the different parts of the Steamship platform and the abstractions it uses.

.. grid::  1 1 2 3

   .. grid-item-card:: **Agents**
      :link: agents/index.html

      Steamship Agents coordinate activites in response to user prompts and directions, often through conversational interfaces.

   .. grid-item-card:: **Packages**
      :link: packages/index.html

      Steamship Packages are cloud-hosted software libraries you can use in any programming environment.

   .. grid-item-card:: **Plugins**
      :link: plugins/index.html

      Steamship Plugins perform specific tasks related to AI, like directly communicating with LLMs.

   .. grid-item-card:: **Data Model**
      :link: data/index.html

      The Steamship Data Model provides a flexible platform for tenancy, isolation, storage, retrieval, and augmentation.
